Business / Client:
The Crocs Book Store - a small independent bookshop in your neighbourhood.

============================

Features

Must Have:
✔ Add a book
✔ search by title
✔ display all inventory
✔ show low-stock alerts.

Nice to Have:
✔ Sort inventory alphabetically
✔ count total books across all titles

CS30 Extension
✔ given a genre, suggest all books in that genre from the inventory

============================

Overview

This program allows users to manage a collection of books. 
You can display all books, add new books, search for specific books, and sort them.

============================

How to Use

Display All Books
Click the Display button to show all books in the table.

Add a Book
Enter the book title, author, genre, and quantity in the input fields.
Click the Add button to add it to the list.

Search for a Book
Type a book title into the search bar.

Sort Books
Use the sort options (dropdown/menu) to organize books alphabetically.

===========================

Notes / Limitations / Known issues

- Must capitalize all the words when adding a book or else it will remain in lowercase
- The program does not resize well
- The horizontal position for the table is not fixed on certain sized screens
- The program currently only has 5 genres but more is able to be added per request
